Color Cognition Game

Welcome to Color Cognition, a fun and challenging game that tests your memory and reflexes. Follow the instructions below to start playing and see how far you can go!

Press A Key to Start

How to Play

Getting Started:

  • Press any key on your keyboard to initiate the game. If you're on a mobile device, tap anywhere on the screen to start.

Gameplay:

  • The game will randomly display a sequence of colored signals.
  • Your task is to repeat the sequence by pressing the corresponding color buttons.

Sequence Expansion:

  • After each successful round, the game will duplicate the sequence and add a new color.
  • Keep repeating the sequences in the correct order to advance to the next level.

Controls

Laptops/Desktops: Press any key to start and click on the color buttons to repeat the sequence.

Mobile Devices: Tap anywhere on the screen to start and tap on the color buttons to repeat the sequence.
